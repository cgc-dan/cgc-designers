Install a custom MyFreeCams Profile design

Under 1 minute install

MyFreeCams is a felxible and fairly easy to use platform with intuitive navigation.

Custom profiles are usually using both CSS and HTML markup and both of them must be added on MyFreeCams

  1. Login to your MyFreeCams account
  2. Edit Profile – link from the left sidebar
  3. Step Four – About Me/ Bio box
  4. Paste the HTML code there
  5. Click the Save button

Install the CSS style markup code too. Follow the suggested path below:

  1. Customize Profile – link from the left sidebar
  2. Paste the CSS code in the text box.
  3. Click the Save button

Now you can refresh your profile page to update those changes.
Note:
– just in case you want to get back to your old profile design, make sure you have a backup copy saved on you device.

Make sure your About Me section is visible
