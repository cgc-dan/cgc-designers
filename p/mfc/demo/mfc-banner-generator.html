How to use...

1. Type the Title of your link in the first field
2. Grab your MyFreeCams tip URL it should look like this https://mfc.im/Username/t
3. Paste your hosted image url in the next field
4. Type the size of your preview image...300 pixels is pretty visible.
5. If you want to resize the image, in the generated code you'll see width='' between quotes you can add any value expressed in pixels or percent values like so: 300 or 100%
Example:

<a title='Tip Me' target='_self' href='http://mfc.im/Username/t'><img width='300' src='your-image-url-here'></a>

<a title='Tip Me' target='_self' href='http://mfc.im/Username/t'><img width='100%' src='your-image-url-here'></a>
