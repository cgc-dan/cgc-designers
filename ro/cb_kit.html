Introducere

Chaturbate este un site dedicat nisei adult

Membri inregistrati pe Chaturbate au optiunea de a edita profilul (bio) personal folosind imagini, HTML basic si HTML inline-css

Etichete HTML acceptate

Daca esti sau nu familiarizat cu platforma Chaturbate sti deja sau afli acum, ca Chaturbate impune anumite restrictii in ceea ce priveste folosirea markup-ului HTML

Tagu-rile HTML acceptate de platforma Chaturbate:

You may use some HTML tags (a p i strong b u ul ol li h1 h2 h3 img font br). Do not block the site navigation or display IP statistics of visitors.

a – tag HTML ce defineste un link
<a href="

p – tag HTML ce defineste un paragraf
<p>

i – tag HTML ce schimba stilul elementului text din default in italic/oblic
<i>

strong – tag HTML ingroasa elementul text
<strong>

u – adauga o linie sub elementul text
<u>

ul, ol, li – defineste o lista
<ul> / <ol> / <li>
<ul> <li>some text or link here</li> </ul>

h1, h2, h3... – defineste elementele text ca reprezentand incept de titlu (heading)

img – defineste elementul imagine
<img src="

font – personalizeaza elementul strict text
<font

br – defineste un salt in line si incheie randul precedent
<br>

Acestea sunt tagurile HTML folosite si acceptate de platforma Chaturbate

Desi p este element block (display:block) am adaugat totusi atributul pentru a instrui browserul in ce metoda sa il afiseze.
Width - pentru a reda cat mai flexibil elementul paragraf pe diferite rezolutii ale browser-ului, am folosit valoarea exprimata in procente (recomandat)
Height - cu atributul auto.
Elementul p are continut, motiv pentru care il numim element parinte. Daca am o imagine in acest element p cu dimensiunea de 300px inaltime, paragraful (fiind elementul parinte) isi va reda in mod automat inaltimea de 340px (atributul auto nu se va schimba in definitia CSS) din cauza valorilor padding adaugate in CSS
Background color. Culoarea de fundal. Desi in majoritatea cazurilor este recomandata mimificarea (optimizarea) css evita sa faci acest lucru deoarece, Editorul va trebui sa "citeasca" fiecare atribut css in parte.

Exemple compatibilitate Editor

background-image: url(image-path) – recomandat
background:#f1f1f1 – nu este recomandat
background: url(image-path) – nu este recomandat

Optional, alte atribute de style se pot adauga

border, font-size, color, font-variant, text-shadow, border-radius, box-shadow, background-image

Sugestii

Evita utilizarea tagurilor h1, h2, h3 ...h6 deoarece Chaturbate va detecta aceste taguri ca display:block float:none clear:both si strica elementul parinte

Evita spatiul intre segmentele de markup deoarece Chaturbate interpreteaza aceste epatii ca br sau p

Evita pe cat posibil folosirea dimensiunilor (width) fixe exprimate in pixeli ca 200px 456px s.a.m.d pentru ca ne dorim cat mai multa flexibilitate (responsive width)
